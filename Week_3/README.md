# Week 3 - Product REST API

- Create Product
- Read All Products
- Read Product by ID
- Update Product
- Delete Product

---

##  Product Fields

- `productId` → Required
- `productName` → Required
- `price` → Required (10000 - 50000)
- `brand` → Required

---

##  API Endpoints

| POST | /products | Create product |
| GET | /products | Get all products |
| GET | /products/:id | Get product by ID |
| PUT | /products/:id | Update product |
| DELETE | /products/:id | Delete product |
