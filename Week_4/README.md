# HTML - HyperText MarkUp Language

<!DOCTYPE html> – Defines the document type and HTML version
<html> – Root element that contains the entire webpage
<head> – Stores metadata, title, and linked files
<title> – Sets the webpage title shown in the browser tab
<body> – Contains all visible webpage content
<h1> to <h6> – Used for headings of different sizes
<p> – Defines a paragraph of text
<a> – Creates hyperlinks to other pages or websites
<img> – Displays images on the webpage
<br> – Inserts a line break
<hr> – Creates a horizontal line separator
<div> – Used to group and organize sections of content
<ul> and <ol> – Create unordered and ordered lists
<li> – Represents list items inside lists
<table> – Creates tables to display data
<form> – Used to collect user input through forms
<input> – Creates input fields like text boxes and buttons
<button> – Creates clickable buttons on the webpage
